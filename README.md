# 🏥 VoxCare

VoxCare is an AI-powered Smart Hospital Handover System designed to improve communication between healthcare professionals during shift changes.

## Features

- 🩺 Smart patient handover
- 🎙 Voice recording support
- 🤖 AI-generated patient summaries
- 🚨 Critical patient alerts
- 📊 Live patient monitoring dashboard
- 👨‍⚕️ Separate Nurse and Doctor portals

## Technologies Used

- HTML5
- CSS3
- JavaScript
